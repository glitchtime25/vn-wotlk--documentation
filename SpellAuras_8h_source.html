263 typedef void(Aura::*pAuraHandler)(bool Apply, bool Real);
264 // Real == true at aura add/remove
265 // Real == false at aura mod unapply/reapply; when adding/removing dependent aura/item/stat mods
266 //
267 // Code in aura handler can be guarded by if(Real) check if it should execution only at real add/remove of aura
268 //
269 // MAIN RULE: Code MUST NOT be guarded by if(Real) check if it modifies any stats
270 // (percent auras, stats mods, etc)
271 // Second rule: Code must be guarded by if(Real) check if it modifies object state (start/stop attack, send packets to client, etc)
272 //
273 // Other case choice: each code line moved under if(Real) check is mangos speedup,
274 // each setting object update field code line moved under if(Real) check is significant mangos speedup, and less server->client data sends
275 // each packet sending code moved under if(Real) check is _large_ mangos speedup, and lot less server->client data sends
