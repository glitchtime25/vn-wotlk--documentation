uint32  locId,
uint32  linkKind,
Team  team,
bool  inDB = true 
)

Adds a given graveyard link to the currently active graveyard links. Optionally add it into the database too.

Parameters
idThe id of a graveyard. This is an index to the graveyard in sWorldSafeLocsStore which holds data from WorldSafeLocs.dbc
locIdThe id of a location. This is either an area id or a map id based on the value of linkKind
linkKindThe kind of the graveyard link to be added. Kind is either GRAVEYARD_MAPLINK or GRAVEYARD_AREALINK
teamThe team that is allowed to use this graveyard link. Can be TEAM_BOTH_ALLOWED, HORDE or ALLIANCE.
inDBTrue if the given graveyard link needs to be added to the database. False otherwise.
Returns
Whether or not the link was added (False only if the link was already added added).
